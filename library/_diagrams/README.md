# Diagram Assets

This folder stores non-inline visual assets for the learning library.

## Recommended Subfolders
- generated: benchmark charts generated from scripts.
- assets: manually created images and annotated visuals.

## Current Policy
- Use Mermaid inline for most flowcharts in markdown files.
- Add files here only when inline diagrams are not enough.

## Naming Convention
- Use kebab-case names.
- Include topic prefix, for example:
  - transformers-latency-vs-context.png
  - rag-retrieval-quality-comparison.png
