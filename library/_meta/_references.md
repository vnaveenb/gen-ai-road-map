# Shared Glossary and References

## Glossary
- Attention: mechanism that weights token interactions based on query-key similarity.
- KV cache: cached key/value tensors used to speed autoregressive decoding.
- RAG: retrieval-augmented generation, combining retrieval with generation.
- Hybrid retrieval: combining dense vector retrieval and lexical retrieval.
- Reranker: a second-stage model that reorders retrieved candidates.
- Guardrail: policy or technical control that constrains risky model behavior.
- Eval: repeatable test to measure quality, safety, or regressions.
